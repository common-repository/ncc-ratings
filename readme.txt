== Description ==

Using this plugin you can allow users to rate your blog post and therefore see which content users like.

== Installation ==

1. Upload `ncc_ratings.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the NCC Ratings Settings in your Administration Panel
4. Enable the Widget through the Appearance --> Widgets tab.
